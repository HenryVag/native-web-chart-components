# React & React Native Chart Components

This repository provides reusable PieChart and LineChart components for both **React** and **React Native** applications.  
The goal is to create lightweight and easy-to-use components with minimal dependencies, making them simple to integrate into any project.

## Installation

### React
1. Clone the main branch of the repository
2. Import the desired component from src/web/

### React Native

1. Clone the main branch of the repository
2. Install react-native-svg with `bun install react-native-svg` or `npm install react-native-svg`
3. Import the desired component from src/web/
